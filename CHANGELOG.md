# Changelog

## 0.1.2 — 2026-06-19

- Curve/Arc теперь разрываются нативным методом SketchUp `Edge#explode_curve`.
- Добавлена команда Select Open Ends с красными крестиками свободных концов.
- Добавлена команда Highlight Gaps с красными линиями ближайших разрывов.
- Диагностические маркеры находятся в отдельных группах и не меняют исходные контуры.
- После автообновления показывается явный результат в строке состояния SketchUp.
- Новые кнопки добавляются в открытую панель через hot-reload без обязательной переустановки RBZ.

## 0.1.1 — 2026-06-19

- Исправлена передача геометрии верхнего уровня и внутри групп в Flatten и Make Faces.
- Исправлена критическая ошибка `intersect_with` в Make Faces.
- Flatten снова перемещает вершины на выбранную мировую Z.
- Преобразование кривых и дуг в отрезки включено по умолчанию в Break To Segments.

## 0.1.0 — 2026-06-19

- Добавлены Break To Segments, Flatten Edges To Z и Make Faces.
- Добавлены безопасные Undo-операции, лимиты, предупреждения и русские отчёты.
- Добавлены меню Extensions, панель инструментов и минималистичные иконки.
- Добавлены автоматические тесты и ручные сценарии SketchUp.
- Добавлены проверка GitHub Releases, нативное уведомление, транзакционное обновление с rollback и hot-reload без перезапуска для обычных обновлений логики.
- Добавлен автоматический release workflow с RBZ и SHA-256 manifest.
