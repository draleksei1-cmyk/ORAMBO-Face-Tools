# Changelog

## 0.1.0 — 2026-06-19

- Добавлены Break To Segments, Flatten Edges To Z и Make Faces.
- Добавлены безопасные Undo-операции, лимиты, предупреждения и русские отчёты.
- Добавлены меню Extensions, панель инструментов и минималистичные иконки.
- Добавлены автоматические тесты и ручные сценарии SketchUp.
- Добавлены проверка GitHub Releases, нативное уведомление, транзакционное обновление с rollback и hot-reload без перезапуска для обычных обновлений логики.
- Добавлен автоматический release workflow с RBZ и SHA-256 manifest.
